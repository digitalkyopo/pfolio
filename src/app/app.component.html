PROJECTS

InTransition (NVTSI: Operation Reboot)

A mobile-app/web-app developed for the National Veterans Transition Services Inc's Operation Reboot program. This was a team project done with my SSF cohort. It allows users to register themselves and measure their transition progress as well as find resources and follow guidelines through the process.

see demo github

Talkman App

A mobile app created with Ionic hybrid app framework. This app is a nod to Home Alone 2, where Kevin used his Talkboy to fool others by recording phrases and playing them. Talkman allows the user to input a phrase and play it at different speeds for a bit of fun.

see demo github

YOLO Stock

A web app that allows users to search stocks and check their values. This app was made with Angular framework and utilizes the Alpha Vantage API for the stock data.

see demo github

The Breadcrumbs App

A hybrid mobile app that locates your current position on a map and allows you to drop a "breadcrumb" pin on your location. This was created using the Ionic framework and utilizes the Google Maps API for the map being displayed.

see demo github
